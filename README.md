# Blog Pessoal

Projeto de blog pessoal feito durante as aulas do bootcamp de desenvolvimento web full stack da Generation Brasil.

## Stack

No back-end:
* Java;
* MySQL & PostegresSQL;
* Spring Boot;
* NodeJS;
* Deploy com Heroku.

[No front-end](https://github.com/petry078/blogPessoalAngular/):
* HTML, CSS & JavaScript;
* TypeScripe;
* Angular;
* Bootstrap;
* Figma;
* jQuery;
* Popper.JS.
